# Entire

An [Omarchy](https://omarchy.org/) theme with wallpapers from [Entire](https://entire.io/projects).

![Desktop showcase](backgrounds/entire-4.png)

## Wallpapers

All 5 wallpapers are from Entire's website. Check them out at:
**https://entire.io/projects**

## Included

- Hyprland config
- Waybar, Wofi, Walker
- Alacritty, Foot, Kitty, Ghostty
- Neovim, Zellij
- VS Code theme
- Mako notifications
- GTK styling
- Btop, Chromium themes

## Install

```bash
omarchy-theme-install https://github.com/ZeroiJ/Entire.git
```
